# @nextclaw/ui

Official NextClaw Web UI package.

This package is consumed by `nextclaw` and also published for standalone integration/development use.

The reusable chat primitives now live in `@nextclaw/agent-chat-ui`. This package keeps the Nextclaw host shell, presenter/store wiring, config pages, and product-specific application integration.

## Install

```bash
npm i @nextclaw/ui
```

## Development

```bash
pnpm -C packages/nextclaw-ui dev
pnpm -C packages/nextclaw-ui build
```
